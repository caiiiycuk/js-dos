js-dos

js-dos was designed from the ground up to be easily installed and used to get your DOS program up and running in browser quickly.

emulators-ui

emulators-ui is a default set of components that used to build js-dos player ui. You can resuse them to build your custom DOS player.

emulators

emulators is a core of js-dos, it have standartized api and can run emulation in different environments.
