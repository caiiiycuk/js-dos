DOS Sokol

DOS Sokol uses same codebase as worker and direct backends. This is native version of emulator backend, it can be compiled to run on PC. You can compile it with make:

Run
make sokol

Sokol version is used to develop DOS emulator backend. Read more hot to develop it in Contributing guide.
